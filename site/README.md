# ГОША РУБЧИНСКИЙ — Онлайн-магазин (динамическая версия)

## Требования
- PHP 8.0+
- MySQL 8.0+
- Apache / Nginx с поддержкой PHP
- Локальный сервер: XAMPP, OpenServer, Denwer или встроенный PHP-сервер

## Установка

### 1. Настройка базы данных
Откройте phpMyAdmin или выполните в консоли:
```sql
mysql -u root -p < database.sql
```

### 2. Настройка подключения
Откройте `config/db.php` и измените параметры:
```php
define('DB_HOST', 'localhost');
define('DB_USER', 'root');    // ваш пользователь MySQL
define('DB_PASS', '');        // ваш пароль MySQL
define('DB_NAME', 'gosha_rubchinskiy');
```

### 3. Размещение на сервере
Скопируйте папку `site/` в корень веб-сервера (например `htdocs/` для XAMPP).

### 4. Запуск
Откройте в браузере: `http://localhost/`

## Структура файлов
```
site/
├── config/
│   └── db.php          — настройки БД + генератор SVG-заглушек
├── includes/
│   ├── header.php      — общая шапка (навигация, счётчик корзины)
│   └── footer.php      — общий подвал + popup
├── api/
│   ├── cart_add.php    — добавить в корзину (POST JSON)
│   ├── cart_update.php — изменить количество (POST JSON)
│   ├── cart_remove.php — удалить из корзины (POST JSON)
│   └── order_place.php — оформить заказ (POST)
├── css/
│   └── styles.css      — все стили
├── js/
│   ├── index.js        — меню, popup
│   └── slider.js       — Glide.js инициализация
├── index.php           — главная страница
├── catalog.php         — каталог с фильтром и сортировкой
├── product.php         — карточка товара (?id=N)
├── cart.php            — корзина
├── login.php           — вход
├── register.php        — регистрация
├── logout.php          — выход
├── database.sql        — схема БД + демо-данные
└── .htaccess           — настройки Apache
```
